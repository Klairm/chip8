# chip8 PS4
Chip8 interpreter in C

A simple Chip8 interpreter written in C for educational purposes

## TODO:
- Implement a file selection menu ( considering imgui )

## Usage
```bash
./chip8 <ROM>
```

F1 - Reload ROM

F2 - Increase speed 

F3 - Decrease speed

ESC - Exit

## Keyboard
| Original keyboard | Equivalent |
| -------------   | ------------- |
|  1	2	3	C       | 1 2 3 4      |  
|  4	5	6	D       |  Q W E R      |
|  7	8	9	E       |  A S D F      |
|  A	0	B	F       |  Z X C V       |
