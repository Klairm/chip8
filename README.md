# chip8 PS4
Chip8 interpreter in C++ ported to PS4 

A simple Chip8 interpreter written in C++ for educational purposes

## Keyboard
| Original keyboard | Equivalent |
| -------------   | ------------- |
|  1	2	3	C       | 1 2 Y T      |  
|  4	5	6	D       |  Q W E R      |
|  7	8	9	E       |  A S D F      |
|  A	0	B	F       |  Z X C V       |
