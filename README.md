# chip8
Chip8 interpreter in C

A simple Chip8 interpreter written in C for educational purposes

## Building
```bash
cd chip8/
chmod +x install.sh 
./install.sh
```
Note: You can manually build the program just using: 
```bash
cd chip8/
make
```

## Usage
```bash
./chip8 <ROM>
```

F1 - Reload ROM

F2 - Increase speed 

F3 - Decrease speed

ESC - Exit
